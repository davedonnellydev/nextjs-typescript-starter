## Quick Start

1. **Clone and install:**
   ```bash
   git clone <your-repo-url>
   cd nextjs-typescript-starter
   npm install
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your values
   ```

3. **Start development:**
   ```bash
   npm run dev
   ```

4. **Run tests:**
   ```bash
   npm test
   ```
