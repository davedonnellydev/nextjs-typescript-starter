## Quick Start

1. **Clone and install:**
   ```bash
   gh repo clone davedonnellydev/nextjs-typescript-starter
   cd nextjs-typescript-starter
   npm install
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your values
   ```

3. **Start development:**
   ```bash
   npm run dev
   ```

4. **Run tests:**
   ```bash
   npm test
   ```
